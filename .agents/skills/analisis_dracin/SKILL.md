# Skill: Analisis DRACIN

Ketika skill ini aktif, baca file referensi berikut **sebelum** melakukan analisis mandiri:

- [`references/analisis_lengkap.md`](references/analisis_lengkap.md) — Laporan analisis lengkap terakhir
- [`references/arsitektur.md`](references/arsitektur.md) — Ringkasan arsitektur & stack

## Cara Menggunakan

1. Baca referensi di atas terlebih dahulu
2. Bandingkan dengan kondisi kodebase saat ini (file mungkin sudah berubah)
3. Identifikasi apa yang sudah diperbaiki vs masih bermasalah
4. Laporkan temuan baru jika ada
5. Update file referensi jika ada perubahan signifikan
